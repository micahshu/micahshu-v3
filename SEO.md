1. **`freelance web developer Fort Collins`** — stevethewebsiteguy.com uses "freelance" in his about copy but has no page targeting this phrase. focowebdesign.com doesn't use it at all. Google currently returns Upwork and generic directory pages in these results — a clear gap.

3. **`affordable web design Fort Collins`** — juicy-ideas.com uses the word "affordable" but doesn't appear to have a dedicated page optimized for this exact phrase. thriveengine.com uses it too, but as an agency they're underperforming on buyer intent signals for this modifier.

5. **`WordPress developer Fort Collins`** (dedicated page) — both Steve and Mike build WordPress sites, but neither appears to have a dedicated page targeting "WordPress developer" as a standalone service/keyword. This is a page Micah could own cleanly.

6. **`full stack developer Fort Collins`** (freelancer context) — ikonicdev.com targets this, but as an agency. No individual freelancer has a page for it.

7. **`Loveland web design` / `Loveland web developer`** — no Fort Collins freelancer has a Loveland-specific page. The search results show agency aggregators and out-of-town generics. Clean opportunity.

## Section 5 — Berthoud Analysis (Final Verdict)

**Do not build a keyword strategy around Berthoud terms. Volume is effectively zero.**

Evidence: Searching "Berthoud web design" and "Berthoud web developer" returns only programmatic city-page generators (limeup.io, ddaone.com, webengine.io, isadex.com, jaymehta.co, jessicaleighwebdesign.com). None are actually based in Berthoud. Their presence confirms the keyword gets crawled but tells you nothing about real demand — they generate these pages for thousands of cities automatically.

**What Berthoud IS good for:**
- Google Business Profile primary address (Berthoud/Larimer County)
- "Based in Berthoud, serving Fort Collins and Northern Colorado" — natural copy on About/Contact pages
- Local directory citations (Berthoud Chamber, Larimer County business listings)
- Schema markup (`LocalBusiness`, `serviceArea`)

The GBP with a Berthoud address will likely outrank generic city-page generators in the local map pack for "web developer near me" queries originating from Berthoud, Loveland, and the I-25 corridor — without needing any dedicated page content.
